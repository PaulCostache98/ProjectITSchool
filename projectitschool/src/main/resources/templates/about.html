A brief description of this Project

Below you will find a few pointers describing what this project is about.

A Personal Computer Shop

Simulating a Computer Shop

Users will be able to browse Computers, add them to Carts and buy them. Furthermore they will be able to check their Purchase History, their Account Summary and Delete their account if they choose. Admins will be able to edit/add users and towers as well as remove them. Admins are also able to do the same actions users can.
